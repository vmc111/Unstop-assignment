## Getting Started (Run Locally)

1. Clone the repo
   ```bash
   git clone https://github.com/vmc111/Unstop-assignment.git
   cd Unstop-assignment
   ```
2. Install dependencies
   ```bash
   yarn
   # or: npm install / pnpm install
   ```
3. Start the dev server
   ```bash
   yarn dev
   ```
4. Open the app
   - Local: http://localhost:5173
